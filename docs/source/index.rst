Welcome to spikeanalysis's documentation!
=========================================

spikeanalysis is a pure Python approach to analyzing `Phy` style **in vivo** electrophysiology data.
It uses a variety of classes for preparing stimulus data and spike data, for analyzing these data, 
and for plotting these data. 

Check out the :doc:`usage` section for installation instructions :ref:`install <installation>`
